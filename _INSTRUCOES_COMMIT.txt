-----------------------------------------------------

Abra o terminal/PowerShell na pasta do projeto:

  cd C:\BTGTicketsAPI

Inicialize o repositório Git:

  git init
  git config user.name "Murilo Molina"
  git config user.email "murilo.molina@gmail.com"

-----------------------------------------------------
PASSO 3: Fazer Commit
-----------------------------------------------------

Adicione todos os arquivos:

  git add .

  - Guias de instalação e testes funcionais
  - Configuração segura via variáveis de ambiente

-----------------------------------------------------

Se você tem um GitHub/GitLab/Bitbucket:
