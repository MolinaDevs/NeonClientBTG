🚀 COMO USAR:

1. BAIXAR:
   - Baixe este projeto como ZIP
   - Extraia em uma pasta

2. ABRIR NO VISUAL STUDIO:
   - Abra o arquivo: BTGTicketsAPI.csproj
   - Restaure os pacotes NuGet

3. RODAR:
   - Pressione F5
   - Acesse: http://localhost:5000/

4. TESTAR:
   - http://localhost:5000/              ← Página inicial
   - http://localhost:5000/teste-simples.html  ← Testes
   - http://localhost:5000/swagger       ← API docs
